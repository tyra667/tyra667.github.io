ABOUT DASHR

DashR is a free browser-based platformer game created by Tyra667. The game was built entirely with vanilla HTML5, CSS and JavaScript — no game engine, no framework, no external dependencies. Just open your browser and play.

THE GAME

DashR is a precision platformer where speed and movement mastery are everything. Your character can run, jump, perform an 8-directional dash, and chain wall-jumps to reach seemingly impossible platforms. Each death respawns you instantly at the start of the level — no checkpoints, no lives. Just you, the timer, and the goal.

The game features two visual worlds: the sunlit Meadow (Levels 1–5), filled with brambles, trees and rolling hills, and the dark Iron Caverns (Levels 6–10), where teleport portals, faster enemies and tighter layouts will test your reflexes to the limit.

FEATURES

  • 10 handcrafted levels across 2 themed worlds
  • 8-directional dash with directional wall-jump combo
  • Moving platforms, patrolling enemies, and teleport portals
  • Per-level best time tracking and total run timer
  • Full title screen with audio settings, key rebinding, and character dressing room
  • 10 character color presets including an animated Rainbow mode
  • Procedurally generated background decorations (trees, flowers, cave crystals)
  • Retro chiptune sound effects via the Web Audio API
  • Mobile-friendly with virtual joystick and buttons

TECHNOLOGY

DashR is built with zero external dependencies. Everything — physics, rendering, audio, input handling, level design, procedural decoration — is written from scratch in a single HTML file.

  • HTML5 Canvas API for all rendering
  • Web Audio API for procedural chiptune SFX
  • Touch Events API for mobile gamepad
  • Vanilla JavaScript — no libraries, no frameworks
  • Hosted on GitHub Pages — free, fast, and always available

PLAY NOW

The game runs in any modern desktop browser. No account needed, no download, completely free.
